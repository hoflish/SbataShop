# SbataShop - Ecommerce Progressive Web Apps

SbataShop is React and Node.js based eCommerce platform. Allows creating a Progressive Web Apps.

Built with:

- Node.js v8.9
- React v16
- Redux
- Express
- Babel
- WebPack 4
- MongoDB
